https://zippia-jobs-filipegallodev.vercel.app/test/jobs

## Description
Fictional job website.

## ⚙️ Features
- Filter by company name
- Filter by jobs posted max 7 days ago

## Front-End guide
- npm install
- npm run dev
- open in browser

## Back-End guide
- npm install
- npm install -g ts-node (if you don't have)
- npm run dev
